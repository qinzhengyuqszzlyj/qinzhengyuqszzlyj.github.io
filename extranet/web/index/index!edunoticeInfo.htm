闵行区第四届“数据驱动大规模因材施教(智慧教育项目)”申报通知

发布时间: 2023-09-15

各中小学、幼儿园、特殊学校、职业学校、直属单位：

现启动闵行区第四届“数据驱动大规模因材施教(智慧教育)”项目的申报，具体通知如下：

1.申报时间：2023年9月15日至9月30日中午12点（平台开放窗口期）

2.登录地址：

https://jiaoshi.mhedu.sh.cn/eias/home.html?moduleCode=007

3.申报内容：请各校自行下载“数据驱动大规模因材施教”申报书（详见附件1），并依据学校智慧教育发展的需求与教师自身探索的主题，参考“智慧教育项目图谱”（详见附件2），逐一填写、自主申报。

4.申报过程:申请人（即项目负责人）在线填报、提交申报表后，由各学校单位审核人（一般为校园长或信息化分管校园长）登录平台逐一审核本校（园）申请的所有项目（具体操作办法详见附件3）。

5.立项结果：闵行区教育局、教育学院将组织专家对项目加以评审，立项分为区级、校级两类项目。

6.注意事项：

（1）学校若为首次申报课题，请在申报之前联系技术人员增添学校账号；

（2）凡有该项目在研的负责人不得申报，采取一年一结、当年度完成的管理办法；

（3）在申报过程中的技术问题，请联系技术人员（联系电话 邓柯 15111149616）；“科研平台系统”微信群）

（4）在申报过程中如遇专业问题，请与各学段科研员联系。


闵行区教育学院科研中心

2023年9月15日
