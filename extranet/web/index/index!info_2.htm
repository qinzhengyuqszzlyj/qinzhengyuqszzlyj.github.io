初中、高中地理集中教研：教学评保持一致，落实核心素养

发布时间 : 2023-09-18   作者（来源）: 周光明   阅读 : 724

3/8
←
→

第3周，初中、高中地理集体教研活动分别在上海市莘光学校、七宝中学附属鑫都实验中学举行。主要聚焦作业设计与考试评价、高阶思维的培养、教学评保持一致性，落实核心素养等话题，安排了多位教师的主题发言。

活动正式开始前，教研员周光明老师宣布了地理学科中心组名单并颁发聘任证书，让中心组的教师明确自身肩负的引领、示范、辐射职责。


两位初中地理教师的发言，主要是结合各年级的考试结果分析进行思考。

七宝中学附属鑫都实验中学的沈爱花老师分享的内容是“指向核心素养的初中地理命题与评价”，主要结合六年级期末质量检测的命题过程，提醒教师们教学中需要重视素养和能力的培育。

七宝三中的陈焱飞老师分析了“初中地理学业考试和跨学科案例分析的评价要求”，结合学生答题表现的详细剖析，提醒教师的日常教学要以德育为先，夯实基础，重视地理工具，链接真实生活，开展实践活动，不断提高学生解决问题的能力。


两位高中地理教师的发言主要是结合小课题的研究成果进行分享。

上海市闵行区实验高级中学的青年教师刘雪晴老师的发言是“学生创新思维发展视角下的高中地理课堂话语研究”，通过AI智慧教育数字化平台，对自身课堂过程的师生行为进行观察分析，提出教师的课堂设问要能够制定不同的思维层次目标，并通过评价激发学生创新思维、高阶思维的发展。

莘庄中学付慧娟老师的话题是“高中地理教学中开展思维结构评价的实践研究”，围绕高中地理教学内容开展思维结构评价，使理论研究与课堂教学实践相结合，在实践中反思，在反思中完善，不断提升学生的思维品质。


结合地理单元作业设计市级评比中的具体情况，地理教研员周光明老师就“教学文本的规范和语言锤炼”进行了详细的讲解。他对教学文本排版提出了细致的要求，指出要学会借鉴高质量试题和公开发表、出版资料中的文字表述，不断锤炼学习单、作业和命题的语言表达，做到简练、准确、流畅、优美、有趣，强调一份好的教学文本应当具备系统、简约、人文的特征，也鼓励大家持续阅读，积极参与评课交流，重视规范表达。

周老师还从教研工作的定位、理念支持、内容梳理和组织策略等方面对新学期教研计划与教研思路进行全面解读，为大家指明了本学期地理教研工作的重点。

七宝中学柳英华老师对第六届闵行区地理骨干基地学习任务与要求做了具体布置。


开学各项工作繁忙，本次初高中的地理教研活动节奏紧凑、主题鲜明、内容丰富，教师们在一起深入探讨、分享智慧、凝聚共识，为本学期各项工作的谋篇布局做好准备，拓宽了地理教师的视野，工作思路变得更加清晰。
