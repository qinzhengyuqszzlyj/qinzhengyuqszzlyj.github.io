以德润心，助力学生健康幸福成长——闵行区第六期“优秀教师培育工程”心理学科专场展示活动通讯

发布时间 : 2023-11-15   作者（来源）: 朱凤   阅读 : 948

3/8
←
→

2023年11月14日，闵行区第六期优秀教师培育工程心理学科专场展示活动在七宝中学附属鑫都实验中学举办，主题为：以德润心，助力学生健康幸福成长。闵行区教育学院副院长董学平、导师华东师范大学心理与认知学院教授席居哲、实验西校校长章志强、区教育学院校发中心俞琳、鑫都中学副校长沈勍、浦东新区心理学科带头人洋泾中学徐琳老师等出席活动。七宝中学党委副书记鞠瑞利主持活动。

闵行区教育学院教研员陈滢老师和七宝鑫都实验中学周嘉老师分别进行了《魔镜！魔镜！》和《有一点“担心”》现场教学，随后陈滢老师做《精准教学，以研促教：初中生社会情感能力课程建设的思考》的汇报，周嘉老师深入介绍了情绪主题单元课程开发与实施案例。两位老师分别从纵向性别差异的社会情感能力课程整体开发到情绪主题为例横向的一次单元设计的探索，共同构建儿童青少年心理健康的防护网络。

沈勍副校长代表学校致欢迎辞并介绍了学校的办学特色和心理健康教育经验。章志强校长分享了与闵行区教育学院心理学科合作申请的“女生社会情感能力培养课程开发”研究项目的进展及其经验。徐琳老师认为两节课的背后都有调研、技术的支持，通过数据实现精准教学，希望未来能够加强区域间的交流和相互学习。无论从区域或学校的整体构架，还是心理学科的专业领域，闵行区致力于为学生筑起心理安全防线的同时，也关注到部分甚至个体学生的个性化需求。

席居哲教授高度认可本次活动，认为“站位高”，立足国际前沿，青少年社会情感能力的培育与智力能力的培养相辅相成，关乎国家未来发展的命运；“立意高”，充分体现了“立德树人、启智润心”的育人目标；“学理高”，虽然心理学科没有统一纲要，但项目研究组基于各方需求，整合资源，以研促教，呈现了精彩纷呈且对学生有指导实效的课堂。正高级教师、心理特级教师鞠瑞利老师对教学课程探索给予好评，同时在课堂节奏和活动运用等细节处理上给出建设性的意见。

董学平副院长做领导讲话。感谢导师对学员的精心指导和培养，感谢承办学校对本次活动的大力支持。本次活动展示了学员的风采，达到了项目研修的预期目标，成果亮眼。教育学院将依托优秀教师培育工程项目、新一轮心理学科种子基地等，建设一支优秀心理教师团队，促进课程的构建研发、教科研的开展实践，在闵行区教育高质量发展的进程中，发挥心理学科以德润心，助力学生健康幸福成长的作用。

席居哲名师工作室学员，闵行区第七期心理学科带头人、骨干教师、心理骨干后备教师、闵行区初中心理专职教师约100人参加了活动。
